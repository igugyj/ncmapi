# NCMAPI

BetterNCM plugin that exposes a local HTTP API compatible with lx-music's [open API spec](https://lyswhut.github.io/lx-music-doc/desktop/open-api).

Forked from [mo-jinran/Taskbar-Lyrics](https://github.com/mo-jinran/Taskbar-Lyrics) — all rendering/taskbar features removed, repurposed as a pure API service.

refer - - [lx-music Open API](https://lyswhut.github.io/lx-music-doc/desktop/open-api)
